# Topics covered
### Core Java Servlets:
- Basic and advanced servlet concepts, handling HTTP requests and responses, servlet lifecycle, and practical examples.

#### JavaServer Pages (JSP):
- JSP syntax, directives, scriptlets, and integrating with Servlets for dynamic web content.

### Hibernate ORM:
- Introduction to Object-Relational Mapping, configuration, CRUD operations, mapping types (One-to-One, One-to-Many, Many-to-One), and Hibernate annotations/XML mapping files.

### JDBC with Servlets:
- Database integration in servlets, connection setup, performing SQL operations, and practical JDBC code.

### Spring (Intro level):
- Dependency Injection basics and simple DAO patterns using Spring framework.

### Web Application Mini-Projects:
- Examples like banking application, CRUD apps, e-commerce prototypes.
